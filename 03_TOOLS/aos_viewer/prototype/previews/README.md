# 备用美学方向归档

> 2026-06-23 评估后归档，作为备用设计思路保留

## 背景

在 v0.4 基础上探索了三个独有美学方向，经评估后决定保留当前主流设计方案（普适性优先），三个方向归档备用。

## 三个方向

| 方向 | 文件 | 核心隐喻 | 适用场景 |
|------|------|---------|---------|
| A 赛博档案室 | a_cyber_archive.html | 数字档案室，文件即档案 | 强规范感项目 |
| B 终端有机体 | b_terminal_organism.html | 有生命的终端，规范是基因 | 极客向项目 |
| C 东方留白 | c_oriental_void.html | 现代水墨，留白是力量 | 文人向项目 |

## 评估结论

- 三个方向风格属性过重，受众局限性较强
- 当前主流方案虽模板化，但适配绝大多数普通用户
- 主题功能属长期规划，暂不纳入现阶段考量
- 当前阶段在现有主流基础设计思路上优化打磨即可

## 后续使用

- 本项目未来上线主题功能时可参考
- 其他项目需要独特视觉语言时可复用
- 每个方向均为自包含 HTML，可直接独立使用
