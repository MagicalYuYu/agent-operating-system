# 示例项目

一个游戏本地化示例项目，展示 AOS 项目目录的标准结构。

## 项目说明
- 源语言：English
- 目标语言：简体中文
- 项目类型：游戏本地化

## 目录结构
```
_example_project/
├── AGENTS.md       ← 项目专属配置（Agent 自动加载）
├── README.md       ← 项目基本信息（本文件）
├── STATUS.md       ← 项目状态跟踪
└── lang/           ← 翻译文件（示例）
    ├── en/         ← 英文原文
    └── zh-cn/      ← 中文翻译
```

## 注意
这是一个示例项目，目录名以 _ 开头。实际项目请使用项目真实名称作为目录名。
