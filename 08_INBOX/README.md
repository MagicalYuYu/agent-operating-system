# AOS v1.1.0 — 08_INBOX

外部输入入口，所有外部输入必须先进这里。

## 应放什么

- 从外部抓取的网页原始内容（`raw/`）
- 迁移采集的原始数据（`migration/`）
- 外部 API 返回的原始响应
- 待分类处理的外部文件

## 禁止放什么

- 已处理并入库的知识（应放 09_REFERENCE/）
- 项目源代码（应放 01_PROJECTS/）
- 运行时状态（应放 04_MEMORY/）

## 分层规则

```
08_INBOX/
├── raw/                           ← 原始抓取内容
│   └── {YYYYMMDD}/                ← 按日期分层
│       └── {source_slug}.md       ← 原始全文（保留可追溯性）
└── README.md
```

## 处理流程

1. 外部输入先进 `08_INBOX/raw/{YYYYMMDD}/`
2. 经过 WKIS（网页知识入库 Skill）或迁移流程处理后
3. 结构化内容入库至 `09_REFERENCE/`
4. 原始内容保留在 `08_INBOX/raw/` 供追溯

## 注意

- `raw/` 必须存储完整原始文本（保留可追溯性），不做大幅删减
- 原始层与知识库层分离，原始层保留全文，知识库层做结构化整理
- 迁移过程中的临时文件（如 `migration/`）按需归档或清理
