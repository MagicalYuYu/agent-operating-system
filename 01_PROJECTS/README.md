# AOS v1.0 - Projects 目录

项目隔离存放区，每个项目一个独立子目录。

## 应放什么
- 每个项目一个子目录：01_PROJECTS/{project_name}/
- 每个项目必须包含：
  - AGENTS.md — 项目专属配置（Agent 自动加载）
  - README.md — 项目基本信息
  - STATUS.md — 项目状态跟踪

## 禁止放什么
- 非项目内容（工具、知识、日志等）
- 知识内容的副本（只允许引用路径，如 09_REFERENCE/web/xxx.md）

## 项目级 AGENTS.md
每个项目目录下的 AGENTS.md 是该项目的专属配置，包含项目约束、敏感信息（端口/接口/路径）、关联项目、知识引用。Agent 根据上下文自动识别当前项目并加载对应配置。

## 项目内标准结构

每个项目目录下建议包含以下子目录（按需创建，非强制）：

```
01_PROJECTS/{name}/
├── AGENTS.md               ← 项目专属配置（必须）
├── README.md               ← 项目基本信息（必须）
├── STATUS.md               ← 项目状态跟踪（必须）
├── docs/                   ← 项目文档（讨论记录、开发规划、设计文档）
├── src/                    ← 源代码/工作文件
├── config/                 ← 项目配置
└── reference_links.md      ← 知识引用路径（不复制内容）
```

- `docs/`：存放项目相关的讨论文档、开发规划、设计文档等 MD 文件
- `src/`：存放实际工作文件（代码、翻译文件、资源等）
- `config/`：存放项目专属配置文件
- `reference_links.md`：引用 09_REFERENCE/ 中的知识路径，禁止复制内容

## 新建项目
创建新项目时必须完成 6 步初始化流程，详见 AGENTS.md 中的"新建项目流程"章节。缺少 AGENTS.md 的项目视为未完成初始化。
