## 通用坑点

### 1. Trae 无 hooks（架构限制）

**现象**：无法实现"每轮自动生效"。ponytail 在 Trae 上只能是 Skill（按需调用）
或 Rule（始终生效），无法像 Claude Code 那样有 PreToolUse 钩子自动检查阶梯。

**应对**：Skill 的 description 中写明触发条件。Agent 在每次代码任务开始时应检查
是否需要调用。
