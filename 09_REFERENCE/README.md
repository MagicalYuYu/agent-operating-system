# AOS v1.1.0 — 09_REFERENCE

唯一参考知识库入口，知识只存一份，禁止重复。

## 应放什么
- system/ — 系统级参考（AOS设计文档、Trae官方指南等）
- web/ — Web知识（从网页抓取入库的结构化知识）
- _index.md — 知识索引（所有条目的可搜索索引）

## 禁止放什么
- 知识内容的副本（其他位置只能引用路径）
- 项目专属数据（应放 01_PROJECTS/）
- 临时文件

## 入库规则
- 使用 WKIS (SK_0001) 入库
- 入库前检查 _index.md 防重复
- 已存在文件不可静默覆盖
- 每个知识文件旁生成 metadata.json

## 引用方式
其他位置引用知识时只写路径，不复制内容：
```markdown
参考：09_REFERENCE/web/{slug}.md
```
