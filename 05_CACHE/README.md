# AOS v1.1.0 — 05_CACHE

纯临时数据存放区，可随时删除，不可存放重要数据。

## 应放什么

- 工作过程中的临时计算结果
- 生成过程中的中间产物
- 调试时产生的临时文件
- 操作过程的 Checkpoint 文件（如发布前准备工作的进度记录）

## 禁止放什么

- 重要数据（此目录可随时删除）
- 最终交付文件（应放 07_EXPORTS/）
- 长期保留的参考知识（应放 09_REFERENCE/）
- 运行时状态（应放 04_MEMORY/）
- 日志文件（应放 06_LOGS/）

## 注意

- 此目录内容**可随时删除**，不可存放任何重要数据
- 操作完成后应将确认有效的内容迁移到正式目录
- 临时文件超过 7 天的建议清理
- Checkpoint 文件（如 `aos_release_1.1.0_checkpoint.md`）在工作完成后应清理或归档
